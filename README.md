# 🔐 Password Generator

A simple and beginner-friendly **Password Generator** built using **Python**. This project allows users to generate random passwords with different difficulty levels and custom password lengths.

---

## 📌 Features

- 🔢 Generate passwords with **Numbers Only**
- 🔤 Generate passwords with **Letters Only**
- 🔡 Generate passwords with **Letters + Numbers**
- 🔒 Generate **Strong Passwords** (Letters + Numbers + Symbols)
- 📏 Choose your own password length
- 🔄 Generate multiple passwords without restarting the program
- ❌ Handles invalid input

---

- `random` module
